# RL-2026

Репозиторий с домашними заданиями и материалами семинаров по курсу **Reinforcement Learning**.

## Описание

В этом репозитории собраны Jupyter Notebook'и с реализациями и экспериментами по основным алгоритмам обучения с подкреплением. Материалы используются для выполнения домашних заданий, разбора семинаров и практики работы с RL-алгоритмами.
